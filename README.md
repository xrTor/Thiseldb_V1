## הוראות התקנה:
הקימו שרת php/mysql  בXAMPP   
האתר נגיש בכתובת  
http://localhost/thiseldb  
תיקית האחסון של האתר  
C:\xampp\htdocs

## התקנה
י mysql צרו דאטה בייס ב בשם media  
ובצעו ייבוא של הקובץ    
media.sql
